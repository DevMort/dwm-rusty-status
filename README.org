* dwm Rusty Status

[[./example.png]]

** Project Description
Shows a system information in the bar of [[https://dwm.suckless.org/][dwm]].

** Features
+ Extensible
+ Simple
+ Few lines of code
+ Light in resources

** Usage
Put this in your .xinitrc or somewhere that autostarts your programs.
#+begin_src bash
dwm-rusty-status &
#+end_src
